Go by Example - IT

Go è un linguaggio di programmazione open source ideato per la creazione di software semplice, veloce e efficiente.

Go by Example - IT è una introduzione pratica a Go che utilizza programmi di esempio commentati con adeguate spiegazioni. Dai un'occhiata al primo esempio o cerca nella lista intera qui di seguito.

Quella che vedi è la traduzione italiana del progetto Go by Example (gobyexample.com), realizzata dalla community italiana di golang (golang-it).
